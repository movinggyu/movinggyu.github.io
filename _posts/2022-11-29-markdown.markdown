### Markdown이란?
- 일반 텍스트로 서식이 있는 문서를 작성하는 방법
- 블로그 포스팅을 할 때 Markdown을 이용해서 할것
- 다음과 같은 문법들이 있다.

# Header
## Header
### Header
`#, ##, ###` 을 문장 앞에 붙여 사용


*Italic*
_Italic_
`*...*, _..._` 을 문장 앞에 붙여 사용


**Bold**
__Bold__
`**...**, __...__` 을 문장 앞에 붙여 사용


~Strikethrough~
`~...~` 을 문장 앞에 붙여 사용


- Unordered List
* Unordered List
`-...,*...` 으로 사용


1. Ordered List
2. Ordered List
`1. ...` 으로 사용


```python
def say():
    print("Hello Markdown!")
```
역따옴표 3개로 블럭을 만들어 코드블록을 만들 수 있음.
